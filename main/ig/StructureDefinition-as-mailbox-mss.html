Data Type Profile: AS Mailbox MSS Datatype Profile

Official URL: https://interop.esante.gouv.fr/ig/fhir/annuaire/StructureDefinition/as-mailbox-mss Version: 1.1.0-snapshot-3
Active as of 2024-09-12 Computable Name: AsMailboxMSSProfile

Datatype profile créé à partir ContactPoint dans le cadre de l’Annuaire Santé pour décrire les boîtes aux lettres du service de messagerie sécurisée de santé (MSSanté) rattachées aux professionnels et aux structures.

Usage:

  • Use this DataType Constraint: AS Organization Profile, AS Practitioner Profile and AS PractitionerRole Profile
